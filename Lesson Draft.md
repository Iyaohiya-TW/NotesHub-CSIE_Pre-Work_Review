## Quick Review Structure

```Markdown
## Topic Name

### One Sentence Definition

### Why Do We Need It?

### Core Idea

### Key Characteristics

### Advantages

### Disadvantages

### Common Use Cases

### Comparison

### Things People Often Confuse

### Things You MUST Remember
```

#### Topic Name
With some overview/introduction.

#### One Sentence Definition
Definition.

#### Why Do We Need It?
Context/Motivations.

#### Core Idea
Mechanism.

#### Key Characteristics
Help indentify the concept/tech. Clues for when to use it.

#### Advantages
Advantages.

#### Disadvantages
Disadvantages.

#### Common Use Cases
Use cases.

#### Comparison
Comparison.

#### Things People Often Confuse
Run through common blindspots.

#### Things You MUST Remember
AT LEAST run through this if you don't have time.
